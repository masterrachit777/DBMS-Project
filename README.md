# DBMS-Project (Complete)
DBMS Project - Educational Institutions Database

## Run the project

In your Terminal

- first step
  ```bash
  ~ npm install
  ```

- Then to run this project in your localhost
  ```bash
  ~ npm start
  ```
  (Must have node.js installed on your system)
